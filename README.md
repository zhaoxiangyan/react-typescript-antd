# 管理后台

> - 整合typescript和ant design组件库，支持redux进行数据流管理
> - 尝试react hook语法
> - 使用react-intl实现国际化


## 框架构成

- Webpack v4
- React v16
- ReactRouter v5
- Redux v4(react-redux + redux-thunk)
- AntDesign v4
- Typescript
- React-intl v4

## 开始使用

### 安装依赖

### `yarn `or` npm install`

### 启动服务

### `yarn start `or` npm start`

### 启动测试模式

### `yarn test `or` npm test`

### 打包布署

### `yarn build `or` npm run build`
